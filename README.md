## 🌟 Overview
**AuraDocs** is a comprehensive, modern, and lightning-fast PDF Toolkit. It provides a stunning, responsive interface to effortlessly manipulate your PDF files without ever compromising on quality.

## 🛠️ Features
- **Merge Documents**: Combine multiple PDF files into one seamless document.
- **Split & Extract**: Extract specific pages from your document.
- **Organize Pages**: Rotate, reorder, and delete pages visually.
- **Convert Files**: Convert Word, PowerPoint, and Excel files into clean PDFs.
- **Security**: Add custom watermarks to protect your intellectual property.

## 🚀 Built With
This application uses a modern technology stack to deliver blazing fast performance:
* **Backend**: Java 17 & Spring Boot
* **Frontend**: HTML5, Vanilla JS, and Tailwind CSS for rapid UI development
* **PDF Processing**: Apache PDFBox for robust document manipulation
* **Deployment**: Docker multi-stage builds hosted on Render

## 💻 Running Locally

1. Clone the repository
```bash
git clone https://github.com/ARYAN-SINGH-10/AuraDocs.git
```

2. Build the Tailwind CSS
```bash
npm install
npm run build:css
```

3. Run the Spring Boot application
```bash
./mvnw spring-boot:run
```
The application will be available at `http://localhost:8080`.
